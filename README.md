 # 🍕 Pizza Sales Analysis

## 📋 Project Overview
End-to-end data analyst portfolio project analyzing 
pizza sales data using MySQL and Power BI.

## 🛠️ Tools Used
- **MySQL Workbench** - Data storage and SQL analysis
- **Power BI Desktop** - Interactive dashboard
- **Excel/CSV** - Raw data source

## 📊 Dataset
- **Rows:** 48,620 pizza sales records
- **Period:** January 2015 - December 2015
- **Columns:** 12 (pizza_id, order_id, pizza_name, 
  category, size, price, date, time etc.)
